## Bank Loan Analysis Dashboard

A complete end-to-end Finance Analytics project developed using Power BI to analyze bank loan data, identify lending trends, monitor loan performance, and generate business insights through interactive dashboards and visualizations.

## 📌 Project Overview

This project focuses on analyzing financial loan data using Power BI.
The objective of this dashboard is to help financial institutions understand:

Loan approval trends
Total funded amount
Loan repayment performance
Good vs Bad loans
Interest rate analysis
Customer demographics
Regional loan distribution
Debt-to-income ratio insights

The project demonstrates the complete data analytics workflow including:

Data Collection
Data Cleaning
Data Transformation
Data Modeling
Data Merging using INNER JOIN
Data Visualization
Business Insight Generation

## 📂 Dataset Information

The dashboard was created using two finance-related datasets.

Data Processing Steps:
Imported two financial datasets into Power BI
Cleaned missing and duplicate values
Standardized columns and data formats
Performed data transformation using Power Query
Merged both datasets using INNER JOIN
Created relationships and data model
Built KPIs and calculated measures using DAX
Designed interactive dashboard visuals
## 🔄 Data Cleaning & Transformation

The following data cleaning steps were performed:

Removed null values
Removed duplicate records
Corrected inconsistent data types
Standardized date formats
Renamed columns for better readability
Filtered unnecessary fields
Handled missing customer information
Created calculated columns and measures
## 🔗 Data Merging Process

The two datasets were combined using an INNER JOIN based on common fields.

Why INNER JOIN?

INNER JOIN was used to:

Match related records from both datasets
Remove unmatched data
Improve data accuracy
Create a unified financial dataset for analysis
This helped generate accurate loan and customer insights.
